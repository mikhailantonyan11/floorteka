## Инструкция по запуску и публикации

### 1. Запуск сайта у себя (локально)
1) Откройте PowerShell.
2) Перейдите в папку проекта:
   ```
   cd "C:\Users\user\Desktop\САЙТ ФЛОРТЕКА"
   ```
3) Установите зависимости (делается один раз, если уже ставили — можно пропустить):
   ```
   npm install
   ```
4) Запустите сервер:
   ```
   npm start
   ```
5) Откройте в браузере:
   ```
   http://localhost:3000
   ```

Чтобы остановить сервер: в терминале нажмите `Ctrl + C`.

---

### 2. Обновление сайта для других пользователей (Render)
Render берёт обновления с GitHub. Поэтому нужно отправить изменения в GitHub.

1) В PowerShell, в папке проекта:
   ```
   cd "C:\Users\user\Desktop\САЙТ ФЛОРТЕКА"
   git add .
   git commit -m "update site"
   git push
   ```
2) Render автоматически запустит новый деплой.
   - Если авто‑деплой отключён:  
     Render → ваш сервис → **Manual Deploy → Deploy latest commit**.
3) После статуса **Live** сайт обновится по той же ссылке.

---

### 3. Где взять ссылку на сайт
Render → ваш сервис → **Service URL** (вид `https://...onrender.com`).
Эту ссылку можно открыть на любом компьютере.

---

### 4. Если `npm` или `node` не работает
Установите Node.js LTS: https://nodejs.org  
После установки перезапустите терминал и проверьте:
```
node -v
npm -v
```

---

### 5. Если `git` не работает
Установите Git для Windows: https://git-scm.com/downloads  
После установки перезапустите терминал и проверьте:
```
git --version
```
